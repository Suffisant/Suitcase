# Suitcase

Suitcase - Mobile Travel Organizer
Suitcase is an Android application designed to simplify travel preparations through smart packing lists and inventory management. Developed using a hybrid of Kotlin and Java, this app provides a lightweight and intuitive mobile experience for travellers.

🔍 Overview
The goal of Suitcase is to ensure you never forget an essential item again. Whether you're heading on a business trip or a vacation, the app allows users to create, categorize, and manage packing lists directly from their Android device. It leverages modern Android development practices to provide a smooth, responsive UI.

✨ Features
Dual-Language Core: Seamlessly integrates Kotlin for modern features and Java for robust legacy support.

Smart Checklists: Create and check off items as you pack.

Category Management: Organize items by type (Electronics, Clothing, Toiletries, etc.).

State Persistence: Automatically saves your lists so you can pick up where you left off.

Clean UI: A minimalist design focused on utility and speed.

🛠 Tech Stack
Languages: Kotlin & Java

Build System: Gradle

Min SDK: 24

Target SDK: 34

IDE: Android Studio

📂 Project Structure
This project follows the standard Android directory structure:

app/src/main/java: Contains the Java source files.

app/src/main/kotlin: Contains the Kotlin source files.

app/src/main/res: Contains XML layouts, strings, and drawable assets.

build.gradle: Project-wide and module-specific configurations.
